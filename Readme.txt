Projekt zawiera analizę ścieżki zakupowej użytkowników w środowisku e-commerce. Analiza została przeprowadzona na danych testowych z ostatnich 30 dni
od ostatniej instancji w wykorzystanym zbiorze danych przy użyciu SQL (BigQuery). Celem tego projektu było zidentyfikowanie miejsc największej utraty
użytkowników jak i ocena rentowności źródeł ruchu.

W analizie wykorzystano: 

CTE (Common Table Expression) w celach zwiększenia przejrzystości kodu.
Funkcje agregujące i logiczne (CASE WHEN) dla budowy etapów lejka.
Obliczenia matematyczne do wyznaczenia współczynników konwersji (CR). 

Kluczowe wnioski i rekomendacje znajdują się w pliku "Rekomendacje_PL".


The project includes an analysis of the user purchase path in an e-commerce environment. The analysis was performed on test data from the last 30 days
since the last instance in the data set used, using SQL (BigQuery). The aim of this project was to identify the points of greatest user loss
and to assess the profitability of traffic sources.

The analysis used: 

CTE (Common Table Expression) to increase code transparency.
Aggregate and logical functions (CASE WHEN) to build funnel stages.
Mathematical calculations to determine conversion rates (CR). 

Key conclusions and recommendations can be found in the file "Recommendations_En".
